# IndiaElect: Your Interactive Election Assistant 🇮🇳

Welcome to **IndiaElect**, a premium, interactive web application designed to guide citizens through the world's largest democratic process. This project was built for the **Prompt Wars** competition.

## 🌟 Overview

Navigating the electoral process can be complex. IndiaElect simplifies this by providing a step-by-step, interactive guide to the Indian election cycle. From understanding the Model Code of Conduct to verifying your VVPAT slip on Polling Day, IndiaElect ensures every citizen is informed and empowered.

## ✨ Key Features

- **Interactive Election Timeline**: A dynamic, 7-stage vertical timeline detailing the entire Indian electoral process, from the announcement of elections to government formation.
- **Voter Readiness Checklist**: A persistent, interactive checklist ensuring voters have completed crucial steps like checking the Electoral Roll (`voters.eci.gov.in`), submitting Form 6, and preparing their EPIC card.
- **"Namaste!" Assistant Chat**: A floating, culturally-aware chatbot widget providing quick answers to common questions about EVMs, VVPATs, NOTA, and alternative ID proofs.
- **Premium Aesthetics**: Built with a sleek, government-inspired dark mode theme using the colors of the Indian flag (Deep Navy Blue, Saffron, White, and India Green). It features advanced glassmorphism, dynamic gradients, and smooth Framer Motion animations.
- **Multi-Page Experience**: Smooth, state-based routing ensures a clean, uncluttered user experience.

## 🛠️ Technology Stack

- **React.js** (Vite + SWC)
- **Vanilla CSS** (for maximum design control and rich aesthetics)
- **Framer Motion** (for smooth UI animations)
- **Lucide React** (for modern iconography)

## 🚀 How to Run Locally

1. **Clone the repository:**
   ```bash
   git clone https://github.com/ShivrajSS-git/prompt-wars-project.git
   cd prompt-wars-project
   ```

2. **Install dependencies:**
   ```bash
   npm install
   ```

3. **Start the development server:**
   ```bash
   npm run dev
   ```

4. Open your browser and navigate to `http://localhost:5173/`
